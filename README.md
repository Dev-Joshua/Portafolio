# Portafolio
Como portafolio me decidí por hacer una pagina estática, la interfaz esta hecha con HTML5 y CSS3, tiene su header, main y footer. Buen código semántico, dentro del main esta la mayor parte del trabajo, la maquetación la realice con Flexbox y Grid Layout. Próximamente le añadiré diseño responsivo, como también lo transformare en proyecto hecho con React.Js.
